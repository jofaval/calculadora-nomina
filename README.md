### Instalación
[Volver a los contenidos](#contenidos)

Para instalarte el repositorio en local:

Clónalo

```bash
git clone https://github.com/jofaval/calculadora-nomina.git
```

Y posteriormente instala todos los paquetes:

```bash
pnpm install # preferido
# npm install
# yarn install
```
